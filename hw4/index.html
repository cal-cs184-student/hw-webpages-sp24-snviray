  • Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

A shader program is used to describe how to color fragments when rendering.

A vertex shader is used to process vertices and geometry to be used in the fragment shader. The fragment shader ultimately computes the color that the fragment will be when it is rendered based on what the vertex shader outputs.
