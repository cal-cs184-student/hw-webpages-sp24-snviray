Part 1: Bezier Curves with 1D de Casteljau Subdivision

Briefly explain de Casteljau's algorithm

  1. Given n control points, use linear interpolation to insert new points between each consecutive pair of control points with a sliding parameter t.
  1. Recursively do step 1 on the new set of points until subdivision is no longer possible (only one new point remains).
  1. That single point lies on the curve. Repeat the entire algorithm for all values of t in the interval [0, 1].

Explain how you implemented de Casteljau’s algorithm in order to evaluate Bezier curves.

I implemented the recursive step of de Casteljau’s algorithm by running the linear interpolation function on each consecutive pair of control points to find the new point, compiling them into a vector, and returning the vector.

Part 2: Bezier Surfaces with Separable 1D de Casteljau

Briefly explain how de Casteljau algorithm extends to Bezier surfaces

The Casteljau algorithm extends to Bezier surfaces by understanding surfaces to have a 4x4 array of control points that create 4 parallel Bezier curves. Points on the parallel Bezier curves are four control points for another Bezier curve that lies orthogonally to all four Bezier curves. Therefore, the Casteljau algorithm can be used to evaluate all of the Bezier curves that represent a Bezier surface.

Explain how you implemented de Casteljau’s algorithm in order to evaluate Bezier surfaces.

  1. Extend the de Casteljau recursive step I implemented in Part 1 to three dimensions by using Vector3D to do linear interpolation.
  1. Define a helper function that calls the recursive step recursively on a set of control points until one control point remains.
  1. Use the helper function to find four points on the parallel Bezier curves that make up the four control points for the orthogonal Bezier curve.
  1. Use the helper function again using the four points found in step 3 as control points to find the point that lies along the orthogonal Bezier curve. This point also lies on the Bezier surface.

Part 3: Area-Weighted Vertex Normals

Briefly explain how you implemented the area-weighted vertex normals

  1. Iterate over all neighboring triangles by find half-edges associated with them by circling around the vertex using h→twin()→next() until arriving back at the original half-edge.
  1. For each triangle, get it’s normal by getting it from it’s face and add it to a vector of normals. Then normals taken from the face are already “unit face normals (compute via the area vector)”, so I don’t need to manually weight them by area.
  1. Add all the normals together and divide them by the amount of normals in the vector to average them.

Part 6: Loop Subdivision for Mesh Upsampling

Briefly explain how you implemented the loop subdivision and describe any interesting implementation / debugging tricks you have used.

I implemented loop subdivision mostly following the comments on the skeleton code.

  1. I iterated over all existing vertices using for (VertexIter v = mesh.verticesBegin(); v != mesh.verticesEnd(); v++) . I set each v→isNew = false for every vertex in this loop. I also calculated the new position of each vertex and placed it in v→newPosition
  1. I iterated over all edges and calculated the location of the location of the new vertex when the edge is split. I set e→isNew = false for all vertices and also added all the edges to a set of existing edges.
  1. I iterated over the set of existing edges and split each edge in the set. I also set the returned new vertex’s new position to the e→newPosition and set newVertex→isNew = true
  1. In splitEdge, I set the two edges that used to be the split edge isNew = false, and the two new edges became isNew = true . I also set newVertex→isNew = true
  1. I iterated over all edges using for (EdgeIter e = mesh.edgesBegin(); e != mesh.edgesEnd(); e++ ) and if e→isNew and one of the vertices of the edge (e→halfedge()→vertex() or e→halfedge()→twin()→vertex()) is new and one is not, I flipped the edge.
  1. I then iterated over all vertices with for (VertexIter v = mesh.verticesBegin(); v != mesh.verticesEnd(); v++) and set each vertex’s actual position to the position stored in its newPosition
